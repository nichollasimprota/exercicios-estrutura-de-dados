# 📚 Exercicios Estrutura de Dados - Engenharia de Software

Repositório com exercícios desenvolvidos em sala de aula da disciplina de **Estrutura de Dados** do curso de **Engenharia de Software**.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** C++
- **Compilador:** MinGW (GCC para Windows)
- **Editor:** Visual Studio Code

---

## 📂 Exercícios

| Arquivo | Descrição |
|---|---|
| `exercicio.cpp` | Operações matemáticas utilizando funções: soma, subtração, multiplicação, divisão, exponenciação, fatoração e termial |

---

## ▶️ Como Compilar e Executar

### Pelo terminal (MinGW):
```bash
g++ exercicio.cpp -o exercicio
./exercicio
```

### Pelo VS Code:
1. Abra o arquivo `.cpp`
2. Pressione `Ctrl + F5` para compilar e executar

---

## 📖 Conteúdos Abordados

- Funções e chamada de funções
- Protótipos de funções
- Estruturas de repetição
- Recursividade
- Parâmetros com valor default
